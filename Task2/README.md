# Laundry Wallah

## Project Overview
A responsive HTML webpage for **Laundry Wallah / Laundry Mart**, a premier laundry service.

## Pages / Sections
- **Header** — Welcome heading with tagline
- **About** — Introduction to Laundry Wallah / Laundry Mart
- **Image** — Laundry facility photo
- **Services** — Grid of 8 laundry services offered
- **Price List** — Table with service pricing (Rs)
- **Book Now** — Booking form with Name, Email, Phone fields
- **Footer** — Contact information

## Tech Stack
- Pure HTML5 & CSS3 (no frameworks)
- Google Fonts (Playfair Display + DM Sans)

## How to Run
Open `index.html` in any modern browser.
